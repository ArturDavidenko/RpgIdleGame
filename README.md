# Rpg Idle Game

## Introduction

This project is an Idle RPG game currently in the prototyping stage.  
The core idea is to build a RNG-driven RPG system, where gameplay is heavily based on probabilistic outcomes, item-based progression, and automated systems rather than direct player control.  
The player focuses on strategic decisions, while the game resolves outcomes automatically.  

## Current Version
### v0.1 — MVP (Inventory System Prototype)  

Current implementation focuses on the Inventory System prototype, which includes:  

- item management inside inventory grid
- move items
- split stacks
- merge items
- delete items  
  
This is the foundational system for future RPG mechanics.
